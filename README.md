|======= CHESSBOARD 2.0 ===========|
|==================================|

#Description:
-This is a simple program that runs in the console.
-When the program runs it´s pretty straightforward an creates a chessboard.
-The size of the chessbord is set depending of the users input.


#Features:
-I made it able to convert all types of inputs into a integer. 
-Both numbers and words of numbers works properly.
-It doesn´t matter if they are uppercase or lowecase, contains " ", "-" or are empty. 


#How to use:
Start the program in Visual Studio and press "Start" or press ctrl + F5.
